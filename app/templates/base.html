How this page works

{% if request.endpoint == 'admin.dashboard' %}

Monitor key hostel KPIs and use the chart to understand workload trends before taking action in module pages.

{% elif request.endpoint == 'admin.rooms' %}

Create rooms, track occupancy, and delete only empty rooms. Use search and pagination to manage large room lists.

{% elif request.endpoint == 'admin.assign_room' %}

Assign students to available rooms and use smart-match suggestions for same branch/year roommate compatibility.

{% elif request.endpoint == 'finance.fees' %}

Admins generate invoices; students pay pending fees using simulated UPI. Use status filter to track paid/pending quickly.

{% elif request.endpoint == 'finance.expenses' %}

Record operational expenses and review spending history to maintain finance transparency.

{% elif request.endpoint == 'mess.menu' %}

Admins publish weekly meals; students vote like/dislike to improve future menus.

{% elif request.endpoint == 'mess.waste_tracking' %}

Track daily food waste trends to optimize meal planning and reduce unnecessary leftovers.

{% elif request.endpoint == 'complaints.list_complaints' %}

Students raise tickets. Admins assign vendors and move tickets through Open, In Progress, and Resolved stages.

{% elif request.endpoint == 'safety.outpass' %}

Students request out-pass with dates; admins approve or reject and gate token (QR code) is tracked per request.

{% elif request.endpoint == 'safety.sos' %}

Use SOS only for emergencies. It triggers a mock alert to security and rector for rapid response simulation.

{% elif request.endpoint == 'student.dashboard' %}

View your profile snapshot, assigned room, and navigate to service modules like fees, complaints, and safety.

{% elif request.endpoint == 'student.my_room' %}

Check your current room assignment, occupancy, and available inventory items linked to your room.

{% elif request.endpoint == 'student.marketplace' %}

Post items for sale and browse student listings. Use search and pages to find relevant items quickly.

{% elif request.endpoint == 'student.lost_found' %}

Report lost/found items with location details so peers can identify and recover belongings faster.

{% elif request.endpoint == 'admin.analytics' %}

Analyze complaint resolution and operational counts to guide staffing and process improvements.

{% else %}

Use this module page to perform related hostel operations. Submit forms and track updates via flash messages.

{% endif %}
